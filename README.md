\# Student Performance Predictor



A machine learning project that predicts student academic performance using attendance percentage, study hours, and previous scores.



\## Features



\- Data preprocessing using Pandas

\- Linear Regression model

\- Performance evaluation

\- Score prediction for new students



\## Technologies Used



\- Python

\- Pandas

\- Scikit-learn

\- Google Colab



\## Project Workflow



1\. Load dataset

2\. Prepare input and output features

3\. Split data into training and testing sets

4\. Train Linear Regression model

5\. Evaluate model performance

6\. Predict student scores



\## Sample Inputs



\- Attendance

\- Study Hours

\- Previous Score



\## Output



Predicted Final Score



\## Learning Outcomes



This project demonstrates the implementation of a machine learning workflow including data preprocessing, model training, evaluation, and prediction using Python.
